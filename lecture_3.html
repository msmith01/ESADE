Portfolio

Question

What is a portfolio? How do we select a portfolio? If 2 assets have the same returns, which do we prefer?

Portfolio

  • Collection of assets (shares, bonds, derivatives, real estate, etc)
  • Each asset represents a \(\%\) of the total portfolio value. The weight of each asset is represented by \(W_{i}\) with \(\sum_{i=1}^{n}W_{i}=1\)
  • Selection: Want to maximize our return and minimize risk.
  • Given 2 assets with same returns, prefer asset with the lower risk.
  • Risk taking investors demand an extra premium for taking on risk - AKA risk premium

Question (10 minutes)

A financial analyst wants to calculate the expected return of investing in a portfolio of shares whose return is \(1.5\%\) with a \(15\%\) probability, \(5\%\) return with a \(25\%\) probability and \(4\%\) otherwise.

Risk and Return

  • \[E[R_{i}] = p_{1}E[R_{1}] + p_{2}E[R_{2}] + p_{3}E[R_{3}] = \]
  • \[0.15(0.015) + 0.25(0.05) + 0.6(0.04) = 0.0387 = 3.87\%\]

Risk and Return

  • We do not always know the probabilities of expected returns of stocks.
  • We don’t know what the future probabilities look like.
  • What do we do?
    • Take historical returns Yfinance

Risk and Return

  • For a single asset:
    • \[E[R_{i}] = \mu_{i} = \dfrac{1}{T}\sum_{t=1}^{T}R_{t}\]
    • Where \(T\) is the number of historic data points used and \(R_{t}\) is the return in period \(t\).

Question

How do we calculate the return for a portfolio?

Risk and Return

  • For a portfolio with \(N\) assets, the portfolio expected return is the weighted average of the expected return of the individual assets.
  • \(E(R_{p}) = w_{1}E(R_{1}) + w_{2}E(R_{2}) + \dots + w_{N}E(R_{N}) = \sum_{i=1}^{N}w_{i}E(R_{i})\)
  • Where \(p\) stands for portfolio.

The following table shows the expected return from investing in assets A and B under two possible scenarios. (S=1 and S=2) along with their probabilities (1/3) and (2/3).

Compute a portfolios return

\[ E[R_A] = \dfrac{1}{3} 20\% + \dfrac{2}{3}(-3\%) = 4.67\% \\ E[R_{B}] = \dfrac{1}{3} 40\% + \dfrac{2}{3}(0\%) = 13.33\% \\ E[R_{P}] = \dfrac{4.67 + 13.33}{2} = 9\% \]

\[ E[R_{P}] = 0.4 * 4.67\% + 0.6 * 13.33\% = 9.85\% \]

\[ E[R_{P}] = 0.03 * 4.67\% + 0.4 * 13.33\% + 0.03 * 2\% = 7.8\% \]

Risk and Return

  • Risk is related to the dispersion of returns relative to its expected return
  • Previously, we estimated returns by taking the historical mean returns of an asset
  • Now, we do the same but for risk
    • We take the historical variance of an asset’s returns \[ \sigma_{i}^{2} = \dfrac{1}{T-1}\sum_{t=1}^{T}(R_{i,t} - \mu_{i})^{2} \\ \mu \text{ is the average return of asset i} \]
    • The standard deviation is the square root of the variance
    • \[ \text{volatility} = \sigma_{i} = \sqrt{\sigma_{i}^{2}} \]

Portfolio of Three Assets

Expected Return: - The expected return of a portfolio with three assets is the weighted average of the expected returns of each asset: - \(E(R_p) = w_1 E(R_1) + w_2 E(R_2) + w_3 E(R_3)\)

Where:

  • \(E(R_p)\) = Expected return of the portfolio
  • \(E(R_1), E(R_2), E(R_3)\) = Expected returns of assets 1, 2, and 3
  • \(w_1, w_2, w_3\) = Portfolio weights for assets 1, 2, and 3, respectively (where \(w_1 + w_2 + w_3 = 1\))

Variance

  • The variance of a three-asset portfolio takes into account the individual variances and the covariances between the assets:

  • \(\sigma_p^2 = w_1^2 \sigma_1^2 + w_2^2 \sigma_2^2 + w_3^2 \sigma_3^2 + 2 w_1 w_2 \text{Cov}(R_1, R_2) + 2 w_1 w_3 \text{Cov}(R_1, R_3) + 2 w_2 w_3 \text{Cov}(R_2, R_3)\)

Where:

  • \(\sigma_p^2\) = Variance of the portfolio
  • \(\sigma_1^2, \sigma_2^2, \sigma_3^2\) = Variances of assets 1, 2, and 3, respectively

Covariances

  • To compute the covariances:
  • Covariance between Asset 1 and Asset 2:
    • \(\text{Cov}(R_1, R_2) = (R_1 - R_1^{\mu})(R_2 - R_2^{\mu})\)
  • Covariance between Asset 1 and Asset 3:
    • \(\text{Cov}(R_1, R_3) = (R_1 - R_1^{\mu})(R_3 - R_3^{\mu})\)
  • Covariance between Asset 2 and Asset 3:
    • \(\text{Cov}(R_2, R_3) = (R_2 - R_2^{\mu})(R_3 - R_3^{\mu})\)

Standard Deviation

Standard Deviation:

  • The standard deviation of the portfolio is:

  • \(\sigma_p = \sqrt{\sigma_p^2}\)

  • These are the fundamental formulas for calculating the expected return and variance (risk) of a portfolio consisting of two and three assets.

  • As we continue to diversify - we reach the systematic market risk (the risk inherent to the entire market or market segment)
